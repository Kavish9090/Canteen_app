# Canteen App 🍔

A Flutter-based mobile application for managing canteen orders, featuring separate interfaces for students and admins.

## ✨ Features
- **Student App**: Browse menu, add to cart, and track orders in real-time.
- **Admin App**: Manage menu items and fulfill orders.
- **Auth**: Secure login/signup using Firebase.

## 🚀 Tech Stack
- **Frontend**: Flutter & Dart
- **Backend**: Firebase (Firestore, Auth, Storage)
- **State Management**: Provider

## 🛠️ How to Run
1. Clone the repo: `git clone https://github.com/Kavish9090/Canteen_app.git`
2. Install dependencies: `flutter pub get`
3. Add your `google-services.json` (Android) to `android/app/`.
4. Run: `flutter run`
